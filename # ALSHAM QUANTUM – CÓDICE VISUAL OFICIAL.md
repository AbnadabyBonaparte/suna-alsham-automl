# 📘 ALSHAM QUANTUM: THE VISUAL REALITY CODEX (v1.0)
**Classificação:** ULTRA-SECRET // DESIGN ARCHITECTURE
**Objetivo:** Definir os 5 "Universos Visuais" do sistema.

## 🌌 REALIDADE 1: QUANTUM LAB (O Padrão)
**Arquétipo:** O Cientista Visionário (Tony Stark / CERN / Minority Report).
**Conceito:** Um laboratório de pesquisa de alta energia. Tudo é experimental, preciso e leve.

### 🖼️ O Cenário (Background & Atmosfera)
*   **Fundo:** Não é preto sólido. É um gradiente radial profundo (`#000000` para `#051015`).
*   **Elemento Vital:** **Partículas Conectadas**. Uma malha (mesh) interativa feita em Canvas/Three.js onde pontos se conectam quando o mouse passa, simulando átomos ou dados quânticos.
*   **Iluminação:** Luzes de borda (Rim Light) ciano, simulando vidro iluminado por LED lateral.

### 🧬 Elementos de Interface (UI DNA)
*   **Cards:** Vidro de alta transparência (`opacity: 0.1`) com `backdrop-filter: blur(12px)`.
*   **Bordas:** Finas (1px), brilhantes e sólidas. Cor: Ciano Elétrico (`#00FFC8`).
*   **Formas:** Cantos chanfrados (45 graus) em apenas um lado do card, ou arredondados suavemente (`12px`).
*   **Tipografia:** `Orbitron` (Títulos) + `Rajdhani` (Dados).

### ⚡ Comportamento & Animação
*   **Hover:** O card acende de dentro para fora (Glow interno).
*   **Loading:** Círculos giratórios concêntricos (como um reator de arco).
*   **Gráficos:** Linhas de néon com "rastro" (trailing effect).

## 🎖️ REALIDADE 2: MILITARY OPS
**Arquétipo:** O General de Guerra (Pentagon / DEFCON / Call of Duty).

*   **Scanlines:** Uma linha horizontal varre a tela de cima para baixo a cada 5 segundos (efeito CRT sutil).

*   **Gráficos:** Barras sólidas (histogramas). Estilo Radar/Sonar.

## 🛠️ GUIA DE IMPLEMENTAÇÃO TÉCNICA (Para o Agente de Código)

**ATENÇÃO, DESENVOLVEDOR:**
Para alcançar este nível, você **NÃO PODE** usar apenas cores condicionais. Você deve estruturar o `ThemeContext` para retornar um objeto de configuração completo:

```typescript
type RealityConfig = {
  id: 'quantum' | 'military' | 'neural' | 'titanium' | 'ascension';
  colors: { ... }; // CSS Variables map
  geometry: {
    radius: string;
    borderWidth: string;
  };
  assets: {
    noiseTexture: boolean;
    scanlines: boolean;
    particles: boolean;
    gridOverlay: boolean;
  };
  typography: {
    displayFont: string;
    monoFont: string;
  };
  effects: {
    backdropBlur: string;
    boxShadow: string;
  }
}
```

**Instruções de Renderização:**
1.  No `layout.tsx`, crie componentes de Background condicionais: `<ParticleBackground />` (só renderiza se `theme === 'quantum'`), `<TacticalGrid />` (só renderiza se `theme === 'military'`), etc.
2.  Use `data-theme` no `<body>` para injetar as variáveis CSS globais de cores.
3.  Use `framer-motion` para trocar a física das animações (spring stiff para Military, spring soft para Neural).
