Instruções para integração com API

Para conectar este dashboard aos dados reais do sistema SUNA-ALSHAM, configure os seguintes endpoints:

/api/status
Retorna status geral do sistema
{"system_status":"active","total_agents":50,...}
/api/cycles
Contagem de ciclos executados
{"cycles_total":0,"cycles_per_second":0}
/api/performance
Métricas de desempenho atual
{"cpu":0,"memory":0,"network":0,"global":0}
/api/logs/stream
Stream de logs em tempo real
WebSocket: ws://hostname/api/logs/stream
